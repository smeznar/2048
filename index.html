How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

You can also try winning the game with AI.
Select the type of AI selection:
  • Best: plays random games in the background and selects the best one.
  • Best with bias: similar to best but with bias to some moves (predetermined or chosen). TODO
  • Random: plays random moves.
You can also select the number of games played in each direction with best selection types.
When your happy with the configuration press the Start AI button and see how far it can get.
